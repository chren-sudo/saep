分两步执行：

**第 1 步 — 获取 admin Token：**



```bash
curl -s -X POST http://127.0.0.1:8000/api/v1/auth/login/ \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' | python3 -c "import sys,json; print(json.load(sys.stdin)['data']['access'])"
```

返回的 Token 复制后执行第 2 步。

**第 2 步 — 导入 Excel：**



```bash
curl -s -X POST http://127.0.0.1:8000/api/v1/students/import/ \
  -H "Authorization: Bearer <粘贴Token到这里>" \
  -F "file=@students-test.xlsx" | python3 -m json.tool
```

> 确保先在终端 `cd` 到 `students-test.xlsx` 所在目录。
