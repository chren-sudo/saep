# **ui-design.md**

# **学生成果认定与综合测评管理平台**

UI设计规范

版本：V1.0

------

# **一、设计原则**

目标：

打造简洁、易用、符合高校办公场景的管理系统。

关键词：

- 简洁
- 清晰
- 高效
- 易维护

禁止：

- 炫酷动画
- 复杂交互
- 花哨配色

------

# **二、技术方案**

```yaml
Vue3
Element Plus
TypeScript
```

------

# **三、整体布局**

采用经典后台布局：

```text
┌───────────────────────────┐
│ Header                    │
├───────┬───────────────────┤
│ Menu  │     Content       │
│       │                   │
│       │                   │
└───────┴───────────────────┘
```

------

# **四、颜色规范**

主色：

```css
#409EFF
```

Element Plus默认蓝。

成功：

```css
#67C23A
```

警告：

```css
#E6A23C
```

危险：

