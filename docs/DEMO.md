# **SAEP v1.0 演示指南**

## 准备

启动前后端后，打开 `http://localhost:5173`

## 演示顺序

### 1. 学生流程（20250001 / 250001）

1. **登录** → 看到 Dashboard
2. **成果管理** → 新增成果 → 上传附件 → 保存草稿
3. 编辑成果 → 修改信息 → 保存
4. 提交审核 → 确认弹窗 → 状态变为"已提交"
5. **我的成绩** → 查看总分+排名

### 2. 测评小组流程（eval01 / eval123456）

1. **登录** → Dashboard 显示待审核数
2. **成果审核** → 看到待审列表 → 点"审核"
3. Drawer 显示详情 + 附件 → 打分 → 通过
4. 另一条成果 → 退回（填写意见）→ 状态变为草稿
5. 另一条成果 → 驳回（填写意见）→ 状态变为已驳回

### 3. 辅导员流程（counselor01 / counselor123456）

1. **登录** → Dashboard
2. **成果审核** → 看到待终审列表 → 通过
3. **公示管理** → 生成公示 → 发布 → 结束
4. **排行榜** → 切换到对应批次 → 查看排名
5. 导出 Excel

### 4. 管理员流程（admin / admin123）

1. **登录** → Dashboard 显示全局统计
2. **排行榜** → 查看所有班级排名
3. **导出成绩** / **导出成果** → 下载 Excel

## 关键亮点

- 附件预览（图片/PDF 直接在审核 Drawer 中查看）
- 审核记录时间线（退回/驳回原因完整可追溯）
- 竞赛排名法（同分同排名 1,2,2,4）
- 一键 Excel 导出
- 操作日志自动记录
